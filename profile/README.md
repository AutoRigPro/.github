## 🧠 Overview

**Auto Rig Pro** is a professional character rigging add-on for Blender designed to speed up the process of building production-ready rigs for animation. It provides a structured and efficient workflow for creating skeletons, controls, and deformation systems for both humanoid and non-humanoid characters.

The tool focuses on automation and flexibility, allowing users to quickly generate rigs using guided placement systems while still maintaining full manual control when needed. It supports complex character setups including fingers, facial rigs, limbs, and custom structures, making it suitable for a wide range of animation and game development pipelines.

Auto Rig Pro also includes advanced retargeting features, enabling users to transfer animations between different characters and skeleton structures. This makes it especially useful for studios and artists working with motion capture data, reusable animation libraries, or multiple character variations.

The add-on integrates deeply with Blender’s workflow and supports export pipelines for real-time engines, ensuring compatibility with common production environments. It is widely used in character animation, game development, cinematic production, and 3D content creation workflows.
